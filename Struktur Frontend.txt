## API Integration Mapping

### Public Endpoints (No Auth Required)
- GET /api/v1/comics â†’ useComics()
- GET /api/v1/comics/{slug} â†’ useComic(slug)
- GET /api/v1/comics/{id}/chapters â†’ useChapters(comicId)
- GET /api/v1/chapters/{id} â†’ useChapter(id)
- GET /api/v1/genres â†’ useGenres()
- GET /api/v1/search â†’ useSearch(query)

### Protected Endpoints (Auth Required)
- POST /api/v1/bookmarks â†’ useBookmark().add()
- DELETE /api/v1/bookmarks/{id} â†’ useBookmark().remove()
- GET /api/v1/bookmarks â†’ useBookmark().list()
- POST /api/v1/reading-history â†’ useHistory().record()
- GET /api/v1/reading-history â†’ useHistory().list()
- POST /api/v1/comments â†’ useComments().create()
- POST /api/v1/ratings â†’ useRating().submit()
- GET /api/v1/user â†’ useAuth().getUser()
- PUT /api/v1/profile â†’ useAuth().updateProfile()

### Admin Endpoints
- GET /api/v1/admin/dashboard â†’ AdminDashboard
- GET /api/v1/admin/comics â†’ ComicManagement
- POST /api/v1/admin/comics â†’ CreateComic
- PUT /api/v1/admin/comics/{id} â†’ EditComic
- DELETE /api/v1/admin/comics/{id} â†’ DeleteComic
- POST /api/v1/admin/chapters â†’ CreateChapter
- PUT /api/v1/admin/chapters/{id} â†’ EditChapter

## Key Features

### 1. **Responsive Design**
   - Desktop-first approach
   - Breakpoints: mobile (< 768px), tablet (768-1024px), desktop (> 1024px)

### 2. **Image Optimization**
   - Next.js Image component untuk cover & banners
   - Lazy loading untuk chapter pages
   - Progressive loading dengan blur placeholder

### 3. **Performance**
   - Server-side rendering untuk SEO
   - ISR (Incremental Static Regeneration) untuk popular comics
   - Client-side caching dengan SWR/React Query
   - Infinite scroll untuk lists

### 4. **User Experience**
   - Dark/Light mode toggle
   - Keyboard shortcuts untuk reader
   - Reading progress tracker
   - Bookmark sync across devices
   - Continue reading from last position

### 5. **MTL Integration**
   - Real-time translation toggle
   - Multiple translation providers
   - Translation caching untuk performance
   - Language preference persistence

### 6. **SEO Optimization**
   - Dynamic meta tags per page
   - Open Graph tags
   - JSON-LD structured data
   - Sitemap generation
   - Robots.txt configuration
