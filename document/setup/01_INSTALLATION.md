# Instalasi Frontend

## Prerequisites

- **Node.js** 18.x atau lebih tinggi
- **npm** atau **yarn** atau **pnpm**
- **Git**

## Step 1: Clone Repository

```bash
git clone https://github.com/username/Frontend-KomikoID-MTL.git
cd Frontend-KomikoID-MTL
```

## Step 2: Install Dependencies

```bash
# Menggunakan npm
npm install

# Atau menggunakan yarn
yarn install

# Atau menggunakan pnpm
pnpm install
```

## Step 3: Verifikasi Instalasi

```bash
# Check Node version
node -v  # Harus 18.x atau lebih

# Check npm version
npm -v
```

## Checklist Instalasi

- [ ] Node.js 18+ terinstall
- [ ] npm/yarn/pnpm terinstall
- [ ] Repository sudah di-clone
- [ ] `npm install` berhasil
